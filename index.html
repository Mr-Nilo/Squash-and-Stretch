The principle

A simple distortion that brings objects to life

When an object hits something, it briefly squashes. When it moves quickly, it briefly stretches. This tiny exaggeration creates the illusion of weight, flexibility, and physicality — and forms the foundation of every other animation principle.

Squash

When an object hits a surface, it compresses — becoming wider and shorter. This communicates impact, weight, and the deformation of soft materials.

Stretch

When an object moves quickly, it elongates — becoming taller and narrower. This communicates speed, momentum, and the direction of motion.

Constant volume

Core principles

Three rules that make it feel right

Following these three rules separates believable animation from animation that feels off. Break them and motion looks rubbery. Honor them and your work feels alive.

01

Preserve volume

When width increases, height must decrease by the same proportion. The mathematical heart of the principle is scaleX × scaleY = 1. This rule alone is what makes deformation read as physical rather than as melting.

02

Extremes define personality

The amount of squash and stretch you apply communicates what the object is made of. Subtle deformation feels heavy and realistic. Exaggerated deformation feels cartoony and light — a rubber ball stretches more than a bowling ball.

03

Time it precisely

Squash happens only at the moment of impact. Stretch happens only during the fastest motion. The deformation lives for just one to three frames — any longer and the object reads as gum, not rubber.

Real-world examples

Look closely — it's everywhere

From Disney animations to the buttons in modern apps, this principle quietly shapes how we experience motion. Watch each example and notice the deformation at work.

Classic
The bouncing ball
Watch the ball flatten the moment it hits the ground — that's squash. Between bounces, as it picks up speed, it elongates into an egg shape — that's stretch. The textbook demonstration.
Character
The jumping figure
Notice how the body squashes flat on landing and stretches tall mid-air. The same volume-preserving deformation works on any shape — not just a ball.
Interface
The button press
On press, the button gets wider and shorter — a quick squash that signals the click landed. Volume preservation makes the digital element feel physical.
From idea to code

Translating the principle into syntax

The same principle can be expressed with CSS keyframes, JavaScript physics, or a single transform. Each approach has its place — explore them all below.

bounce.css
/* The ball element */
.ball {
  width: 60px;
  height: 60px;
  border-radius: 50%;
  background: #155581;
  animation: bounce 0.8s cubic-bezier(0.215, 0.61, 0.355, 1) infinite;
  transform-origin: center bottom;  /* pivot from bottom */
}

@keyframes bounce {
  0%   { transform: translateY(0) scale(1, 1); }

  /* Stretch — moving fast upward */
  15%  { transform: translateY(-80px) scale(0.75, 1.35); }

  /* Peak — natural shape at top */
  50%  { transform: translateY(-160px) scale(1, 1); }

  /* Stretch — falling fast */
  85%  { transform: translateY(-80px) scale(0.75, 1.35); }

  /* Squash — impact: wide and flat */
  95%  { transform: translateY(0) scale(1.45, 0.65); }

  /* Recovery — return to shape */
  100% { transform: translateY(0) scale(1, 1); }
}
// Squash & stretch with real physics
function getBallScale(y, vy, groundY) {
  const distFromGround = groundY - y;
  const speed = Math.abs(vy);

  let scaleX = 1, scaleY = 1;

  // Squash: close to ground, moving down
  if (distFromGround < 20 && vy > 0) {
    const squashFactor = 0.6;
    const t = 1 - (distFromGround / 20);
    scaleY = 1 - ((1 - squashFactor) * t);
    scaleX = 1 / scaleY;  // preserve volume
  }

  // Stretch: fast vertical motion
  else if (speed > 3) {
    const stretchFactor = 1.4;
    scaleY = 1 + ((stretchFactor - 1) * (speed / 12));
    scaleX = 1 / scaleY;  // preserve volume
  }

  return { scaleX, scaleY };
}

ctx.save();
ctx.translate(ball.x, ball.y);
ctx.scale(scaleX, scaleY);
ctx.arc(0, 0, ball.radius, 0, Math.PI * 2);
ctx.fill();
ctx.restore();
/* The one-liner: button squash on click */
.btn {
  transition: transform 0.1s ease;
  transform-origin: center;
}

.btn:active {
  transform: scale(1.12, 0.88);  /* wider, shorter */
}

/* A character jump with all stages */
@keyframes jump {
  0%   { transform: scale(1.2, 0.8); }                    /* anticipation */
  20%  { transform: scale(0.8, 1.3) translateY(-10px); }  /* launch */
  50%  { transform: scale(0.85, 1.2) translateY(-60px); } /* peak */
  80%  { transform: scale(0.8, 1.3) translateY(-10px); }  /* falling */
  90%  { transform: scale(1.3, 0.75); }                   /* landing */
  100% { transform: scale(1, 1); }                       /* settle */
}
